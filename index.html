Todo

All you have to do after cloning is create a 777 chmoded "uploads" directory at root level.

$ cd Sites
$ git clone git://github.com/Manuite/photobooth.git
$ cd photobooth
$ mkdir uploads
$ chmod 777 uploads

There you go! Have a simply super day!
