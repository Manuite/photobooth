This is my little contribution to this script. It mimics osx's photobooth with the implementation of five different masks.

Todo

All you have to do after cloning is create a 777 chmoded "uploads" directory at root level and two subfolders: "original" and "thumbs".

$ cd Sites
$ git clone git://github.com/Manuite/photobooth.git
$ cd photobooth
$ mkdir -m 777 uploads
$ cd uploads
$ mkdir -m 777 original
$ mkdir -m 777 thumbs

There you go! Have a simply super day!
